//Linux//

Why use Linux on a virtual machine, or use it to hack?
The benefits of using linux, is that it is a open source operating system. This means that you can inspect the code of the software and look what is in it and how it was coded. And see what programs or functions it runs. The OS can be used for many reasons for example: for hacking, servers, embedded digital smart devices, programmers. Another advantage is that most of the linux distro's that those are free of cost. While other(not all) OS system's you might need to pay for license.
While linux is free to download. Because linux is open source it allows users to edit, copy or distribute various aspects of linux -based operating system without violating any copyright law or terms and conditions. This is one of the main reasons hackers why hackers use linux. It is because they can easily develop software used for hacking and penetration testing on linux and they change and edit the operating system for their needs. The OS is very light to run and doesn't require a lot of disk space and ram to run, and is thus easy to run. so it can be easily ran alongside any other operating system like windows and MacOS. Due to Linux being open source and allowing anyone to change and interact with the operating system, it allows people to create an operating system that will specifically made for one purpose, such as, for example, ethical hacking or penetration testing. Linux distro's like this already exist that specifically made for pen testing and checking for security loopholes, and all of them are widely being used by hackers. for this course kali linux would be used. one of the linux distros specially created for hackers and penetration testers.
